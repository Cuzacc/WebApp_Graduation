## Overview
Web App tương tác phục vụ lễ tốt nghiệp có cơ chế Admin đăng nhập kiểm duyệt (Moderation). App sử dụng kiến trúc chuẩn DevOps để phục vụ mục đích đưa vào CV.

## Tech Stack
- Frontend: Next.js + TailwindCSS
- Backend: Python FastAPI
- Database: PostgreSQL
- Hạ tầng: Docker, CI/CD
